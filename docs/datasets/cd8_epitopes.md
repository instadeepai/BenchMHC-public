# CD8 epitopes

## Original Data

### Evaluation

The CD8 epitopes dataset used in BenchMHC originates from the [NetMHCpan-4.1 evaluation data](https://services.healthtech.dtu.dk/suppl/immunology/NAR_NetMHCpan_NetMHCIIpan/).

As detailed in the original paper, this dataset consists of the epitope dataset from
NetMHCpan-4.0[^1], containing epitopes from IEDB that have been validated with T cell assays
capturing IFN-γ, combined with multimer validated epitopes also obtained from the IEDB and
downloaded on 11/04/2020. It was filtered to only contain epitopes of length 8-14 and annotated
with the source protein sequence. The resulting data set, comprising 1,660 epitopes mapped to one
of 52 distinct MHC-I molecules, was downloaded from NetMHCpan-4.1[^2] as provided. The decoys are
generated by sliding windows over the source protein
of the true peptide, with CD8 decoys generated using window lengths ranging from 8 to 14, inclusive.

The evaluation dataset can be found at:

```bash
gs://bench-mhc/data/v0.0.0/evaluation/raw_data/
```
